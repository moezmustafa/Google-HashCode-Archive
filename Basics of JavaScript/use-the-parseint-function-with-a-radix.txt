//in the parseint() we can use the redix to tell the system what type of numbers we are working with 
// in the given example we are using a binary number , that is a base 2 
// the default base for a pasreint() function is base 10 
// to alter that we use redix , the base system in mathematics

function convertToInteger(str) {

  return parseInt(str , 2 );

}

convertToInteger("10011");  // returns 19 
convertToInteger("111001");  // returns 57
convertToInteger("JamesBond"); // returns NaN
